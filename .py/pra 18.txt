"""Write a function in Python that counts the number of “Me” or “My” words present in a
text file “STORY.TXT”.
If the “STORY.TXT” contents are as follows:
My first book was Me and My Family. 
It gave me chance to be Known to the world.
The output of the function should be:
Count of Me/My in file: 3"""


f=open("xyz.txt","r")
x=f.read()
y=x.split()
count=0
for i in y:
    if i in ["Me","My"]:
        count+=1
    else:
        pass

print(count)
